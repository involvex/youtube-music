Choose Your Package Manager

Install using your preferred package manager

# Ubuntu/Debian wget -qO- https://github.com/involvex/youtube-music/releases/latest/download/install.sh | bash # Arch Linux (AUR) yay -S ytmusic-involvex-edition # Fedora/CentOS/RHEL sudo dnf install ytmusic-involvex-edition # AppImage wget https://github.com/involvex/youtube-music/releases/latest/download/ytmusic-involvex-edition.AppImage chmod +x ytmusic-involvex-edition.AppImage

Manual Installation

Alternatively, download and extract the latest .tar.gz release

# Download and extract wget https://github.com/involvex/youtube-music/releases/latest/download/ytmusic-involvex-edition-linux.tar.gz tar -xzf ytmusic-involvex-edition-linux.tar.gz sudo mv ytmusic-involvex-edition /opt/ sudo ln -s /opt/ytmusic-involvex-edition/ytmusic-involvex-edition /usr/local/bin/
3

Run YTMusic Involvex-Edition from your applications menu or terminal
